## About Various modular TV style overlays heavily inspired by 2015 season FIA WEC for iRacing Browser Apps. Main purpose is to recreate a TV like experience for watched sessions or replays in iRacing. You add the modules to different scenes in OBS (or your other favorite capture software) and switch between scenes as you see fit. ## Setup 1. Go to [Settings](#/settings) page if you need to configure the host of the iRacing Browser Apps server and/or FPS. 2. Copy the generated **CLR Browser URL** of the desired module from here or from [Settings](#/settings) page. 3. In OBS add a new **CLR Browser** layer in your scene and paste the **URL** 4. Set **Dimensions** to `Width: 1920` and `Height: 1080` Repeat these steps for other modules or add other scenes with different combinations of modules.
